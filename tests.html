  • 1.1.1
    非文本内容
    A
    是
    找出网页中的非文本验证码后，发现验证码仅提供视觉形式，没有提供听觉或触觉替代方案。对于视障人士，这样的验证码无法使用。
  • 1.3.1
    信息和关系
    A
    是
    二维码图片提供替代文本描述其用途。页面中的二维码应有明确的说明。
  • 1.3.2
    有含义的顺序
    A
    是
    网页内容的顺序可以在html语句中规定，如使用有序列表
  • 1.3.3
    感官特性
    A
    是
    网页上的某些控件（如图标按钮）通过视觉特性（如形状、大小、颜色）来传达信息，也有提供其他感官反馈（如文本描述或语音提示）。
  • 1.3.4
    定位
    AA
    是
    选择“竖向”，测试网页显示正常
  • 1.3.5
    明确输入目的
    AA
    是
    网页上的某些输入框有提供明确的输入目的提示。
  • 1.3.6
    明确目的
    AAA
    是
    在使用标记语言实现的内容中，用户界面组件、图标及区域的用途可以编程式确定。
  • 1.4.1
    颜色用途
    A
    是
    文本颜色不作为传达信息的唯一手段，使用了其他视觉提示。这对色盲和色弱用户尤为重要。
  • 3
    对比度（最小）
    AA
    否
    网页上某些文本与背景的对比度不足，特别是导航栏和搜索栏的文本，无法达到最小对比度要求（4.5:1）。这会对视力较弱的用户和色盲用户造成困扰。
  • 1.4.4
    调整文本
    AA
    是
    文本可以通过浏览器放大功能进行调整，放大至200%仍能正常显示且不影响内容或功能。这对低视力用户非常重要。
  • 1.4.5
    文本图像
    AA
    否
    网页上的横幅广告使用了图像形式的文本，未提供文本的替代信息。图像形式的文本在缩放或使用屏幕阅读器时可能不易读取或理解。
  • 1.4.6
    对比度（加强）
    AAA
    是
    网页上某些文本与背景的对比度很明显，特别是一些按钮和标签的文本，颜色为浅灰色和蓝色。
  • 1.4.8
    视觉呈现
    AAA
    否
    前景颜色和背景颜色无法由用户选择。这对低视力和阅读障碍用户影响较大。
  • 1.4.9
    文本图像（没有例外）
    AAA
    否
    网页上有部分图像形式的文本，这些文本在缩放或使用屏幕阅读器时可能不易读取或理解，且未提供适当的替代文本信息。
  • 1.4.10
    反复滚动
    AA
    是
    网页上存在反复滚动的内容区域，例如横幅广告和推荐内容滚动条，但无异常。
  • 1.4.11
    非文本对比
    AA
    否
    网页上存在多个非文本元素，如图片和图标，其对比度不足，无法确保视障人士清晰辨认。例如，推荐酒店部分的图标和文字对比度不足，难以辨认。
  • 1.4.12
    文本间距
    AA
    否
    网页上的文本间距未达到无障碍标准，导致用户在阅读时体验不佳。例如，页脚的链接文本之间的行间距和段落间距不足，影响阅读体验。
  • 1.4.13
    悬停或焦点内容
    AA
    是
    把鼠标放在网页中，网页会自动弹出一些内容或者信息，悬停的时候可以清除、可以悬停、可以持久有效
  • 2.1.1
    键盘
    A
    否
    网页上的部分功能无法通过键盘接口实现操作，或者在某些操作过程中需要特定时限的键击操作。这导致视障或肢体障碍用户在使用键盘进行导航和操作时会遇到困难。例如，下拉菜单和弹出窗口无法通过键盘完全操作。
  • 2.1.2
    无键盘陷阱
    一级
    否
    在网页上进行键盘导航时，发现某些交互元素（如弹出窗口和下拉菜单）存在键盘陷阱。这意味着用户在使用键盘进行操作时，可能会被困在某个元素中，无法继续进行其他操作。这对依赖键盘操作的用户（如视障或肢体障碍用户）造成了很大的不便。
  • 2.1.3
    键盘（没有例外）
    A
    否
    网页上的部分功能无法通过键盘接口实现操作，这对依赖键盘操作的用户（如视障或肢体障碍用户）造成了很大的不便。例如，下拉菜单和弹出窗口无法通过键盘完全操作。
  • 2.1.4
    字符键快捷键
    A
    是
    网页上不存在字符键快捷键的问题。
  • 2.2.1
    定时可调
    A
    否
    网页上的某些功能具有定时要求，例如图中所示的剩余时间倒计时功能。该功能在一定时间后自动执行操作，对于依赖辅助技术的用户而言，无法灵活调整定时器或延长操作时间，这会对他们的使用体验造成不便。
  • 2.2.2
    暂停、停止、隐藏
    A
    否
    网页上的某些自动播放（轮播图）或自动滚动的内容缺乏暂停、停止或隐藏的功能。
  • 2.2.3
    无定时
    A
    否
    网页上的某些内容存在定时操作，用户未能在规定时间内完成操作将导致内容丢失或页面刷新。例如，页面中存在的计时功能（如预订确认页面的倒计时）没有提供足够的时间供用户完成操作，尤其是对于视障用户或操作速度较慢的用户。
  • 2.2.5
    重新认证
    A
    是
    在网页上，当会话过期时，用户通过重新填写，页面保留着上一次填写的信息。
  • 2.2.6
    超时
    A
    是
    在网页上，当会话超时时，用户收到明确的通知。例如下单支付页面
  • 2.3.1
    闪光三次或低于阈值
    A
    否
    网页中包含一些元素，它们在短时间内闪光超过三次，或者闪光的频率低于一般阈值。这种闪光可能会对有光敏性癫痫或其他光敏性条件的用户造成触发风险。
  • 2.3.2
    闪光三次
    A
    否
    网页中包含一些元素，它们在短时间内闪光不超过三次，都是鼠标停留闪光的。
  • 2.3.4
    交互动画
    A
    否
